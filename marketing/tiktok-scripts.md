# Porfilr — TikTok-native scripts (also Reels / Shorts)

These are RAW and native — not the polished brand ad. TikTok rewards: hook in 2s,
captions on, trending audio, fast cuts, human/founder voice. 15–40s each.
Film on a phone OR have the editor keep it rough — polished = looks like an ad = flops.

POST: same video to TikTok + Reels + Shorts. Batch 2–3 on a free day.
CAPTION KEYWORDS: include "portfolio for [designers/developers/etc.]" — TikTok search = SEO.

---

## 1. POV / relatable pain (best first video)
**Format:** text-on-screen over you (or a screen), trending "oh no" / awkward sound.

[0–2s] ON-SCREEN: "POV: you send a Google Doc as your portfolio"
[2–6s] ON-SCREEN: "…and the client never replies 💀"
[6–12s] cut to a clean Porfilr portfolio on screen — ON-SCREEN: "do this instead"
[12–18s] "A real portfolio site. 10 minutes. No code. Free."
[18–20s] ON-SCREEN: "porfilr.com"

**Caption:** Stop doing this to yourself 💀 free portfolio in 10 min → porfilr.com
#freelance #portfolio #freelancetips #graphicdesign #fyp

**Audio:** a trending "awkward/oh no" sound (search Trending in the audio tab).

## 3. Satisfying build / demo (rewatch magic)

[0–2s] HOOK ON-SCREEN: "Building a portfolio in 60 seconds ⏱️"
[2–10s] screen: pick a template
[10–20s] screen: type name, add work, add a project (sped up)
[20–28s] screen: switch the color, flip to mobile preview
[28–34s] screen: hit publish → the live link pops up
[34–38s] show the finished site — ON-SCREEN: "live. no code."
[38–40s] "porfilr.com"

**Caption:** From nothing to a live portfolio in real time 🤯 → porfilr.com
#satisfying #portfolio #nocode #webdesign #fyp

**Audio:** a fast/satisfying trending sound that "hits" on the publish moment.

## Quick rules (every video)
- Hook in 2 seconds or they scroll.
- Captions ON (most watch muted).
- Use a TRENDING sound — it's the #1 reach lever on TikTok.
- 15–40s. Shorter usually wins.
- Reply to every comment (drives the algorithm + relationships).
- Post consistently > post perfectly. Batch 2–3, schedule, move on.
- Don't make it look like an ad. Rough and human beats polished.
